Ctrl + F5 - отладка
Ctrl + delete - вырезает всю строку
Ctrl + F - поиск элемента, если выделить участок кода мышью, а потом нажать горячую клавижу, то искаться будет выделенный код
Ctrl +  + F - Найти и заменить
Ctrl + D - дублировать
Ctrl + R, A - прогнать все тесты
Cntrl +\, Ctrl + 1 - В просмотре различий показать левое окно
Cntrl +\, Ctrl + 2 - В просмотре различий показать оба окно
Cntrl +\, Ctrl + 3 - В просмотре различий показать правое окна
Cntrl + shift + D - xml комментарий выбранной области/функции кода
